# flyer
## What is flyer?
flyer is a source-based package manager written in [Swift](https://swift.org). It is currently not production-ready, but can already install and remove packages. Feel free to go and test it out.
## Usage
After installing, just type flyer or flyer --help and you're ready to go.
## Contributing
Please follow the following rules when contributing:
- Please do NOT submit AI-generated content. We are planning to implement the "Vouch" code check system; so if you do so, you will be banned from further contributing to the project.
- If you are making minor changes, please explain why they are useful and should be implemented in the PR.
- If you are making major changes, be sure to NOT overwrite existing features.
These rules help us keep the code safe, useful and open for everyone.
